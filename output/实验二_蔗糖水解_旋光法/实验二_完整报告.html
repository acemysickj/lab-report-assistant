实验二 旋光法测定蔗糖转化反应的速率常数

（完整报告：含预习报告与后续报告）

一、实验目的

1. 测定蔗糖转化反应的速率常数和半衰期。

2. 了解该反应的反应物浓度与旋光度之间的关系。

3. 了解旋光仪的基本原理，掌握旋光仪的正确使用方法。

二、实验原理

2.1 反应方程式

蔗糖在水溶液中，在氢离子的催化作用下发生水解（转化）反应，生成等物质的量的葡萄糖和果糖：

$$\mathrm{C_{12}H_{22}O_{11}} + \mathrm{H_2O} \xrightarrow{[\mathrm{H}^+]} \mathrm{C_6H_{12}O_6}(\text{葡萄糖}) + \mathrm{C_6H_{12}O_6}(\text{果糖}) \tag{1}$$

2.2 准一级反应动力学模型

反应体系中水大量过量，其浓度可视为常数；H+ 仅起催化作用，浓度在反应过程中保持不变。因此蔗糖水解反应的速率仅取决于蔗糖浓度 $[S]$，表现为准一级反应：

$$-\frac{\mathrm{d}[S]}{\mathrm{d}t} = k[S] \tag{2}$$

积分形式：

$$\ln[S]_t = -kt + \ln[S]_0 \tag{3}$$

式中：$[S]_0$ 为蔗糖初始浓度；$[S]_t$ 为 $t$ 时刻蔗糖浓度；$k$ 为反应速率常数。半衰期公式：

$$t_{1/2} = \frac{\ln 2}{k} = \frac{0.693}{k} \tag{4}$$

2.3 旋光度与浓度的关系

蔗糖、葡萄糖和果糖均为旋光活性物质。在比旋光度 $[\alpha]$ 和液层厚度 $l$ 固定的条件下，旋光度 $\alpha$ 与浓度 $c$ 成正比：

$$\alpha = k_{\alpha}c \tag{5}$$

设反应开始时（$t = 0$）蔗糖浓度为 $a$，$t$ 时刻已反应量为 $x$，则由式 (1) 知每消耗 1 mol 蔗糖即生成 1 mol 葡萄糖和 1 mol 果糖。各时刻旋光度可表示为：

$$\alpha_t = k_S(a - x) + k_G x + k_F x \tag{6}$$

$$\alpha_0 = k_S a \tag{7}$$

$$\alpha_\infty = (k_G + k_F)a \tag{8}$$

式中 $k_S, k_G, k_F$ 分别为蔗糖、葡萄糖、果糖的比例常数。由式 (6)~(8) 消去浓度项得：

$$\frac{a}{a - x} = \frac{\alpha_0 - \alpha_\infty}{\alpha_t - \alpha_\infty} \tag{9}$$

代入一级反应动力学积分式，得到用旋光度表达的动力学方程：

$$\ln(\alpha_t - \alpha_\infty) = -kt + \ln(\alpha_0 - \alpha_\infty) \tag{10}$$

以 $\ln(\alpha_t - \alpha_\infty)$ 对 $t$ 作图得一直线，由斜率求得速率常数 $k$，再由 $k$ 计算半衰期 $t_{1/2}$。

2.4 比旋光度数据

三种物质在钠光 D 线（589 nm）、20 °C、水溶液中的比旋光度如下：

表 1 相关物质的比旋光度

4.2 关键操作要点

（1）HCl 倒入蔗糖溶液后反应即刻启动，从混合到装样完成的耗时是主要的计时误差来源，操作应连贯迅速，尽量减少起始时间偏差。

（2）样品管装液后须确保无气泡，管外壁擦干后再放入样品室，否则影响光路和读数稳定性。

（3）每次读数需待显示稳定后记录。正值示数按"＋复测"键确认，负值示数按"－复测"键确认。

（4）$\alpha_\infty$ 须在反应彻底完成后测量（55 °C 加热 ≥ 30 min 可确保蔗糖完全水解），且应冷却至室温后再读数。

5.4 实验现象观察

反应过程中旋光度从正值（初始 $+7.01^\circ$）逐步减小，在 25~30 min 区间趋近于零并继续下降。完全反应后 $\alpha_\infty$ 为负值（−3.79°），表明体系由右旋转为左旋。该现象与果糖具有强左旋性（$[\alpha] = -91.9^\circ$）的理论预期完全吻合——随着蔗糖不断水解，左旋性强的果糖不断累积，最终使体系整体表现出左旋特性。

6.2 线性回归分析

根据式 (10)，以 $\ln(\alpha_t - \alpha_\infty)$ 对 $t$ 进行最小二乘线性回归。样本量 $n = 15$，自由度 $\mathrm{df} = 13$，$t_{0.025,13} = 2.160$。回归结果如下：

表 7 线性回归结果汇总
参数拟合值标准偏差95% 置信限
斜率 $a = -k$−0.02226$s_a = 0.00009$±0.00020
截距 $b = \ln(\alpha_0 - \alpha_\infty)$2.427$s_b = 0.002$±0.004
决定系数 $R^2$0.99978——
残差标准差 $s_y$0.0030——

$R^2 = 0.99978$ 极为接近 1，表明 $\ln(\alpha_t - \alpha_\infty)$ 与 $t$ 之间具有极其优良的线性关系，验证了蔗糖水解为准一级反应的动力学假设。残差标准差 $s_y = 0.0030$ 表明数据离散度很小。

6.3 最终结果

速率常数：

$$k = -a = 0.02226\ \mathrm{min}^{-1}$$

$$u(k) = t_{0.025,13} \cdot s_a = 2.160 \times 0.00009 = 0.00020\ \mathrm{min}^{-1}$$

$$\therefore\ k = (0.0223 \pm 0.0002)\ \mathrm{min}^{-1}\quad (\text{95\% 置信度})$$

外推初始旋光度：

$$\alpha_0 = \alpha_\infty + e^{b} = -3.79^\circ + e^{2.427} = -3.79^\circ + 11.33^\circ = 7.54^\circ$$

半衰期：

$$t_{1/2} = \frac{\ln 2}{k} = \frac{0.693}{0.02226} = 31.1\ \mathrm{min}$$

$$u(t_{1/2}) = t_{1/2} \cdot \frac{u(k)}{k} = 31.1 \times \frac{0.00020}{0.02226} = 0.3\ \mathrm{min}$$

$$\therefore\ t_{1/2} = (31.1 \pm 0.3)\ \mathrm{min}\quad (\text{95\% 置信度})$$

七、实验结论（结果讨论）

7.1 实验结果总结

在室温 22.0 ℃、大气压 101 kPa 条件下，采用旋光法测定了蔗糖在 3 mol/L 盐酸催化下水解转化反应的速率常数 $k = (0.0223 \pm 0.0002)\ \mathrm{min}^{-1}$，半衰期 $t_{1/2} = (31.1 \pm 0.3)\ \mathrm{min}$（95% 置信度）。线性回归决定系数 $R^2 = 0.99978$，验证了准一级反应动力学假设。

7.2 与文献对比

文献报道，常温下蔗糖酸性水解的速率常数 $k$ 通常在 $0.01 \sim 0.04\ \mathrm{min}^{-1}$ 量级（具体值取决于温度、酸浓度等条件）。本实验测得 $k = 0.0223\ \mathrm{min}^{-1}$，处于文献报道范围内。若以 $k_\mathrm{lit} = 0.020\ \mathrm{min}^{-1}$（约 0.1 mol/L HCl、20 °C）作为参考值，相对误差为：

$$\varepsilon = \frac{|0.02226 - 0.020|}{0.020} \times 100\% \approx 11\%$$

此误差在可接受范围内，且可部分归因于实验条件（酸浓度、温度）与文献条件的差异。

7.3 误差来源分析

（1）计时误差（系统误差，主要来源）。从 HCl 倒入蔗糖溶液到完成装样、放入旋光仪的过程耗时约 1~2 min，导致计时起点与反应实际起始时刻之间存在系统性偏差。该偏差使得记录的时间值系统性地小于实际反应时间，进而使拟合斜率偏小、$k$ 值偏低。这是本实验最主要的系统误差来源，其影响优先于其他误差因素。

（2）温度波动（偶然误差）。蔗糖水解反应的活化能约 107 kJ/mol，速率常数 $k$ 对温度较为敏感——温度每升高 1 °C，$k$ 约增加 10%~15%。本实验未在样品管外套恒温夹套，室温在 22.0 °C 附近的自然波动（±0.5 °C 量级）可引入可观的偶然误差。

（3）$\alpha_\infty$ 测量误差（偶然误差）。$\alpha_\infty$ 五次平行测量的标准偏差为 0.042°，已属较小。但 $\alpha_\infty$ 代入 $\ln(\alpha_t - \alpha_\infty)$ 时，对后期数据点（$\alpha_t$ 接近 $\alpha_\infty$）的对数值影响相对较大，因为此时 $\alpha_t - \alpha_\infty$ 的绝对值较小。

（4）旋光仪读数误差（偶然误差）。WZZ-2B 自动旋光仪分辨率 0.01°，读数波动对数据点的贡献相对有限，属次要误差来源。

7.4 改进建议

（1）采用快速混合进样方案（如使用可在线加酸的流通式旋光管，或在旋光管内直接混合），使反应起始时刻与计时起点严格同步，从根本上消除计时系统误差。

（2）使用带恒温夹套的旋光样品管，配合高精度循环恒温水浴（控温精度 ±0.1 °C），有效遏制温度波动对 $k$ 值的影响。

（3）将 $\alpha_\infty$ 的测量次数增加至 8~10 次，并适当延长恒温时间至 60 min 以确保蔗糖彻底水解，从而降低 $\alpha_\infty$ 平均值的标准不确定度。

7.5 小结

本实验利用旋光法成功测定了蔗糖水解转化反应的动力学参数，$\ln(\alpha_t - \alpha_\infty)$ 与 $t$ 之间呈优异的线性关系（$R^2 = 0.99978$），充分验证了准一级反应模型。所得 $k = (0.0223 \pm 0.0002)\ \mathrm{min}^{-1}$ 和 $t_{1/2} = (31.1 \pm 0.3)\ \mathrm{min}$ 与文献报道处于同一量级。计时偏差为最主要的系统误差来源，后续可通过改进进样方式和加装恒温装置进一步提高测量准确度。
