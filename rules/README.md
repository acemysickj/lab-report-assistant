# 实验报告格式规范

请在此目录下为报告各章节补充具体规范。文件命名建议：

- `purpose.md` — 实验目的部分的撰写规范
- `principle.md` — 实验原理部分的撰写规范
- `equipment.md` — 仪器与试剂部分的格式规范
- `procedure.md` — 实验步骤部分的撰写规范
- `record.md` — 实验记录部分的规范
- `data-processing.md` — 数据处理与分析的规范
- `discussion.md` — 结果讨论部分的规范
- `questions.md` — 思考题部分的规范
- `format.md` — 整体排版与 HTML 输出格式规范

每个文件可以用自然语言描述该部分的字数、格式、注意事项等要求。Skill 会在生成对应部分时自动遵循。
